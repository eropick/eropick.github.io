# 원의 방정식
$$x^2 + y^2 = r^2$$

기본 출력 위치 : 원점 (0,0) 좌측 상단
=> r에 대한 원점이동 필요
=> $$(x-r)^2 + (y-r)^2 = r^2$$
---

# x좌표 수식

$$(x-r)^2 + (y-r)^2 = r^2$$
$$=>(x-r)^2 = r^2 - (y-r)^2$$
$$=>(x-r) = \pm{\sqrt{r^2 - (y-r)^2}}$$
$$=>x = r \pm{\sqrt{r^2 - (y-r)^2}}$$

---
# 구현
1. y를 0 ~ $$2r$$ 까지 반복하고 x를 0 ~ $$2r$$ 까지 반복한다.

2. y와 r을 넣어서 구한 x의 값 2개를 구해서 실수 오차범위를 정수로 매칭한다.

3. 그 값이 x와 일치하면 별을 찍는다.
