문자를 자음인지 모음인지 체크해나감
-> 문자를 하나씩 담는다.
-> 초성, 중성, 종성 플래그를 체크
-> 종성 플래그가 켜졌을 때 조합함

모든 문자 체크 for 루프(
1. 현재 문자가 자음

-> 초성 있으면 
    -> 중성 체크 있으면 
        -> 다음 문자 패턴 검사
            -> 현재 문자가 끝이면 -> 종성 처리
            -> 다음 문자가 모음이 아니면 -> 종성 처리
    -> 중성 체크 없으면 -> 그대로 초성 체크 가져감


-> 초성 없으면 : 체크

2. 현재 문자가 모음

3. 현재 문자가 자음도 모음도 아님
-> 모든 플래그를 초기화

종성 플래그가 켜졌다
    -> 종성이 없는 경우
    -> 종성이 있는 경우
    결과를 조합형으로 변환
    플래그 초기화
)

결과를 문자열로 변환해서 반환
